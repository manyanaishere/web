HTML

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications.

Types and Explanation of tags

  1. the tags not need to be slashed
    2. : img, input, br, hr, meta
    They are not embraced because they don't explain anything, but just implicate specific function.
  2. the difference between 'br' and 'p'

    3. 'br' can adjust the margin between paragraphs contrast to 'p'. However, you can overcome the limit of 'p' by using CSS language, which elaborate the appearance of web pages. You can put the information of paragraph style in 'p' tag. "style='margin-top:45px'" is the normal expression of CSS.

    As you notice, CSS tag for paragraph margin is applied to this paragraph.

  3. the difference between ol and ul

    4. ol stands for an ordered list, which has the numbers on each component of the list. Otherwise, ul stands for an unordered list, which doesn't have any numbers but only dot on each component.

  4. the 'a' tag's attributes

    5. target : target="_blank" means the new window appears, whereas target="_self" means the same frame turns into the new web page.
